About This Page

This portfolio website is a personal project inspired by the AWS Cloud Resume Challenge. It demonstrates my ability to architect and implement secure, scalable cloud solutions by combining front-end development with back-end services.

Infrastructure Overview

Below is a diagram that illustrates the architecture of this project, including services like S3, CloudFront, DynamoDB, and Lambda, as well as their interactions.

Two Versions of the Cloud Resume Challenge

The Hands-On Journey

My first implementation of the challenge was done manually through the AWS Console, focusing on understanding each service and configuring them step by step. Highlights include:

  • Hosting static website content on S3 with CloudFront as a CDN.
  • Configuring DNS and SSL certificates via Route 53 and ACM for secure access to galazkar.com.
  • Implementing a visitor counter using DynamoDB, Lambda, and JavaScript.
  • Automating deployment with GitHub Actions for CI/CD.

Read more about this approach in my article: The AWS Cloud Resume Challenge.

The Terraform Conversion

To enhance and streamline the solution, I translated the infrastructure into code using Terraform. This version prioritizes automation and maintainability. Key achievements include:

  • Defining reusable Terraform modules for resources like S3, CloudFront, and Route 53.
  • Automating bucket object uploads with CloudFront Origin Access Identity (OAI).
  • Programmatically configuring DNS records in Route 53.
  • Ensuring seamless infrastructure updates using Terraform workflows.

Learn more about this in my article: Infrastructure Evolution: Terraforming My Cloud Resume on AWS.

Key Technologies Used

  • HTML, CSS, and JavaScript for the front end.
  • AWS services: S3, CloudFront, DynamoDB, Lambda, Route 53, ACM.
  • Terraform for Infrastructure as Code (IaC).
  • GitHub Actions/GitLab for CI/CD pipelines.

This project allowed me to explore both manual configurations and Infrastructure as Code, strengthening my expertise in designing production-ready cloud solutions.
